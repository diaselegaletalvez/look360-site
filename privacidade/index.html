Privacidade

Resumo

O Look360 é um aplicativo de orçamentos para profissionais autônomos que respeita totalmente sua privacidade. Nenhum dado pessoal seu é coletado, armazenado ou enviado para servidores externos pelo desenvolvedor.

Quais dados o app guarda?

O Look360 armazena, apenas no seu próprio aparelho, os seguintes dados que você opcionalmente fornece:

  • Seu nome e apelido
  • Seu e-mail (opcional)
  • Seu telefone (opcional)
  • Dados da sua empresa (nome, telefone, e-mail, CNPJ/CPF, chave PIX) — todos opcionais
  • Orçamentos que você cria
  • Senha (em formato criptografado, nunca em texto puro)
  • PIN de segurança (em formato criptografado, nunca em texto puro)
  • Sua chave de API de IA, se você optar por usar o Assistente (guardada no cofre seguro do sistema operacional)

Todos esses dados ficam exclusivamente no seu dispositivo. Se você desinstalar o app, todos os dados são apagados permanentemente.

Dados enviados para terceiros

O Look360 não envia dados para servidores nossos. Não temos servidor.

Quando você opta por usar o Assistente de IA, o conteúdo do seu orçamento é enviado diretamente para o provedor que você escolheu (OpenAI ou Anthropic), usando a sua própria chave de API. Sua chave nunca passa por nós.

Permissões do dispositivo

O Look360 pode solicitar:

  • Face ID / Touch ID: apenas para autenticar o acesso ao app, com seu consentimento explícito.
  • Compartilhamento (Share Sheet): apenas quando você toca em "Compartilhar PDF" para enviar um orçamento.

Cookies, rastreamento e publicidade

O Look360 não usa cookies, não tem rastreadores e não exibe anúncios.

Crianças

O Look360 não é direcionado a crianças. Não coletamos intencionalmente dados de menores de 13 anos.

Alterações nesta política

Podemos atualizar esta política. Mudanças serão publicadas nesta mesma página com nova "Última atualização".
